# Cosmic-Atlas

Cosmic Atlas is a desktop astronomy application built using Python and CustomTkinter.

The project allows users to explore the Solar System through an interactive graphical interface. Users can select planets, view important information about them, see images, and discover interesting space facts.

## Features

-Information about the planets of the Solar System
-Planet images
-Space-themed graphical interface
-Random Space Fact generator
-Interactive desktop application built with CustomTkinter

## Technologies Used

- Python
- CustomTkinter
- Pillow (PIL)
